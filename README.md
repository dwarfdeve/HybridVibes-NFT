# HybridVibes NFT Collection

Premium cinematic 3D hybrid NFTs blending BAYC + Doodles + Lost Dogs + Pudgy Penguins.

## Quick Start
1. Generate images using prompts/ folder
2. Upload images to Arweave
3. Update image links in metadata/
4. `sugar upload`
5. `sugar create-candy-machine`

See deploy.md for full guide.
